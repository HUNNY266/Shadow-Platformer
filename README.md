# 🌑 Shadow Platformer (Unity 2D Game)

A 2D platformer game built using Unity and C#, focused on smooth player movement, level-based progression, and interactive gameplay mechanics.

---

## 🎮 Gameplay Overview

Shadow Platformer is a side-scrolling platformer where the player navigates through levels filled with obstacles, enemies, and collectibles. The game emphasizes responsive controls, level design, and core gameplay systems.

Platformer games typically focus on jumping mechanics, obstacles, and level progression, making them a strong foundation for game development skills :contentReference[oaicite:0]{index=0}

---

## ✨ Features

- 🕹️ Smooth player movement (run, jump, climb)
- 🧗 Ladder climbing system
- 👾 Enemy AI with collision handling
- 💰 Coin collection system
- 🔊 Audio system (jump, coin, death sounds)
- 💀 Death & respawn system
- 📊 Score tracking
- 🎥 Camera follow system (Cinemachine)
- 🎨 2D tile-based level design

---

## 🛠️ Tech Stack

- Unity (2D URP)
- C#
- Unity Input System
- Cinemachine
- Tilemap System
- Particle System
- Audio System
